# Bank Management System in C

A console-based bank management system developed in C, built as part of a structured programming learning journey.

## Description

This program simulates a banking management system in the console.
It allows managing clients and their bank accounts through an interactive menu.

## Project Status

Version 1.0 - first functional console version.

Upcoming versions will include :
- Data saving and loading via files
- Various improvements

The project is currently under development.

## Features

- PIN authentication at startup
- Create, edit and delete clients and accounts
- Deposit, withdrawal and transfer between accounts
- Transaction history
- Client ranking by wealth
- Account total calculation

## Technical Details

- Language : C (C99 standard)
- Compiler : GNU GCC
- IDE : Code::Blocks (Windows)
- No global variables
- No dynamic memory allocation (malloc/realloc)
- Main structures : CLIENT, COMPTE, RESULTAT_RECHERCHE, HISTORIQUE
