# Bank Management System in C 🏦

A **console-based bank management system** developed in C, created as part of a structured programming learning journey. This is the **first functional version**, focused entirely on core banking operations in the console.

---

## Description

This program simulates a banking management system in the console.  
It allows managing **clients** and their **bank accounts** via an interactive menu.  
All operations are handled in-memory, with clear structures and menus to ensure smooth user interaction.

---

## Project Status

**Version 1.0** — first fully functional console version.  

**Planned in future versions:**  
- Persistent data saving/loading via files  
- Code modularization for better maintenance  
- Enhanced validation and error handling  
- Advanced analytics and reports  

---

## Features

- **PIN authentication** at startup for secure access  
- **Client management:** create, edit, and delete clients  
- **Account management:** create, edit, and delete accounts  
- **Transactions:** deposit, withdrawal, and transfer between accounts  
- **Transaction history** for each account  
- **Client ranking** by total wealth  
- **Account total calculation**

## Technical Details

- **Language:** C (C99 standard)  
- **Compiler:** GNU GCC  
- **IDE:** Code::Blocks (Windows)  
- **No global variables**  
- **No dynamic memory allocation** (`malloc/realloc`)  
- **Main structures used:** `CLIENT`, `COMPTE`, `RESULTAT_RECHERCHE`, `HISTORIQUE`  
- Fully interactive console program
